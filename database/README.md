# Database SalonOS

Questa cartella contiene la struttura e la gestione dei dati di SalonOS.

Comprende:

- informazioni dei saloni;
- utenti e ruoli;
- clienti;
- servizi;
- appuntamenti;
- conversazioni AI;
- statistiche.

Obiettivo:

Creare un sistema sicuro dove ogni salone possiede i propri dati separati dagli altri.
