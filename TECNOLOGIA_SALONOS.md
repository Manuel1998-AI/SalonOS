# Tecnologia SalonOS

## Obiettivo

Creare una piattaforma moderna, sicura e scalabile capace di gestire migliaia di saloni.

---

# Frontend

La parte visibile agli utenti.

Responsabile di:

- schermate;
- dashboard;
- calendario;
- chat;
- gestione impostazioni.

Tecnologia prevista:

React / Next.js

---

# Backend

Il cervello dell'applicazione.

Responsabile di:

- utenti;
- saloni;
- prenotazioni;
- permessi;
- comunicazione con AI.

Tecnologia prevista:

Python / FastAPI

---

# Database

Memorizza:

- saloni;
- utenti;
- clienti;
- servizi;
- appuntamenti;
- conversazioni.

Tecnologia prevista:

PostgreSQL

---

# Intelligenza Artificiale

Responsabile di:

- conversazioni;
- comprensione richieste;
- suggerimenti;
- analisi dati.

---

# Sicurezza

Principi:

- dati separati per ogni salone;
- accessi controllati;
- protezione informazioni clienti.

---

# Obiettivo finale

Creare una piattaforma che possa partire da pochi saloni e crescere fino a migliaia senza dover essere riscritta.
